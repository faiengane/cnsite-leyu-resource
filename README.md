# cnsite-leyu-resource

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
仓库内容以静态网页为主，涵盖不同场景下的前端页面资源。  
本仓库不针对任何特定域名或具体网站，仅作为页面文件的存储与版本管理用途。

## 目录说明

```
cnsite-leyu-resource/
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 页面引用的资源文件（图片、样式、脚本等）
└── README.md       # 本说明文件
```

- `pages/` 目录下每个子文件夹或文件代表一个独立页面，命名尽量反映页面用途。
- `assets/` 目录按需组织，与页面文件对应存放静态资源。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开运行。
- 页面可能包含外部资源引用（如字体、库文件），建议使用本地或 CDN 引用方式确保兼容。
- 归档页面不保证与任何在线服务或网站的关联性，仅作为前端代码示例或备份。

## 维护说明

- 欢迎提交 Issue 指出页面展示问题或提出改进建议。
- 如需添加新页面，请放入 `pages/` 目录并在注释中简要说明用途。
- 请勿提交包含敏感内容、恶意代码或违反平台政策的文件。
- 本仓库遵循常规开源协议，具体见仓库根目录 LICENSE 文件（如有）。
